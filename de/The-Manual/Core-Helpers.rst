Kern-Helfer
###########

Helfer (``Helpers``) bei Views entsprechen den Komponenten bei
Controllern. Sie kapseln Code der von vielen Views, Elementen oder
Layouts benutzt wird.

Dieser Abschnitt beschreibt die mit CakePHP ausgelieferten Helper, wie
den Form-, Html-, JavaScript- und RSS-Helper
