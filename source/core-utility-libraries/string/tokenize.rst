8.3.2 tokenize
--------------

``string tokenize ($data, $separator = ',', $leftBound = '(', $rightBound = ')')``

Tokenizes a string using ``$separator``, ignoring any instance of
``$separator`` that appears between ``$leftBound`` and
``$rightBound``.
