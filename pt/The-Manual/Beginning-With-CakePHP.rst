Começando com CakePHP
#####################

Bem vindo ao "Cookbook", o manual para o CakePHP Framewok de aplicação
web para fazer desenvolvimento muito fácil (moleza)!

O manual assume que você tem o entendimento geral sobre PHP e o
entendimento básico sobre programação orientada a objetos (POO). Há
diferentes funcionalidades dentro do framework usando diferentes
tecnologias - como SQL, JavaScript e XML. O manual não tenta explicar
estas tecnologias, somente aquelas que forem usadas no contexto.
