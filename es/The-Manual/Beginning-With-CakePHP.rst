Comenzando con CakePHP
######################

¡Bienvenido al Cookbook, el manual para el *framework* CakePHP que
convierte el desarrollo de aplicaciones web en un juego de niños!

Este manual supone que posees conocimientos generales de PHP y
conocimientos básicos de programación orientada a objetos (POO). Las
diferentes funciones del *framework* utilizan varias tecnologías -por
ejemplo, SQL, JavaScript o XML—. El manual no pretende explicar dichas
tecnologías, sino cómo se utilizan en este contexto.
