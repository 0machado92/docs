Assistants intégrés
###################

Les Assistants (*Helpers*) sont des classes comme les composants pour la
couche de présentation de votre application. Ils contiennent de la
logique de présentation qui est partagée entre plusieurs vues, éléments
ou *layouts*. Cette section décrit chacun de ces assistants intégrés à
CakePHP tels que Form, Html, JavaScript et RSS. Lisez la rubrique
Assistants pour en apprendre davantage sur eux et sur la manière dont
vous pouvez créer vos propres assistants.
