Mise en cache
#############

La mise en cache peut être effectuée à différents niveaux dans une
application CakePHP. Pour plus d'infos, voyez `comment désactiver le
cache du navigateur </fr/view/431/disablecache>`_, `la mise en cache
d'une page entière ou d'un élément </fr/view/213/cache>`_, `la mise en
cache de requêtes </fr/view/445/cachequeries>`_ ou `la fonction cache -
pour mettre en cache n'importe quoi </fr/view/764/Cache>`_.
