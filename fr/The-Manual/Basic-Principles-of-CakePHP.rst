Principes de base de CakePHP
############################

Le framework CakePHP fournit une base robuste pour votre application. Il
peut se charger de chaque aspect, depuis la requête initiale de
l'utilisateur jusqu'au rendu final d'une page web. Et puisque le
framework suit les principes du motif MVC, il vous permet de
personnaliser et d'étendre facilement la plupart des aspects de votre
application.

Le framework fournit également une structure organisationnelle basique,
des noms de fichiers aux noms de table d'une base de données, assurant
consistance et logique à votre application tout entière. Ce concept est
simple mais puissant. Suivez les conventions et vous saurez toujours
exactement où se trouvent les choses et comment elles sont organisées.
